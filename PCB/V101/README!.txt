CCDBusTransceiver_V101

- Gerber files are located in "gerbers" folder.
- Additional informations for each gerber file are in "infos" folder.
- Rendered preview images of the PCB are in "renders" folder.
- PCB board's exact size is 33x20 mm with 1 mm radius on the corners.
- Please leave 8 mil silkscreen linewidth intact.

- IMPORTANT: Some SMD components have soldermask bridges inbetween their pads and themselves. 
  As long as they are manufacturable please keep them. If not then 
  it's okay to remove them.

- Metric drills used:

 	Code  Size        Used     Size (mm)
	------------------------------------
	T01   0.0197inch    15 --> 0.5 mm
	T02   0.0394inch     9 --> 1.0 mm
	------------------------------------
	Total number of drills: 24
